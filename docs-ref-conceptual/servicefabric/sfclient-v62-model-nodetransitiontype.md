# NodeTransitionType enum

type: string

Possible values include: 'Invalid', 'Start', 'Stop'

Possible values are: 

  - `Invalid` - Reserved.  Do not pass into API.
  - `Start` - Transition a stopped node to up.
  - `Stop` - Transition an up node to stopped.
