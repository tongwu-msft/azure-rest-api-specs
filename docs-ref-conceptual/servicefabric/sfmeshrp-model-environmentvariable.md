# EnvironmentVariable

Describes an environment variable for the container.

## Properties
| Name | Type | Required |
| --- | --- | --- |
| [`name`](#name) | string | No |
| [`value`](#value) | string | No |

____
### `name`
__Type__: string <br/>
__Required__: No<br/>
<br/>
The name of the environment variable.

____
### `value`
__Type__: string <br/>
__Required__: No<br/>
<br/>
The value of the environment variable.
