# CreateComposeApplicationDescription

Defines description for creating a Service Fabric compose application.


## Properties
| Name | Type | Required |
| --- | --- | --- |
| [ApplicationName](#applicationname) | string | Yes |
| [ComposeFileContent](#composefilecontent) | string | Yes |
| [RepositoryCredential](#repositorycredential) | [RepositoryCredential](sfclient-v56-model-repositorycredential.md) | No |

____
### ApplicationName
__Type__: string <br/>
__Required__: Yes<br/>
<br/>
The name of the application, including the 'fabric:' URI scheme.

____
### ComposeFileContent
__Type__: string <br/>
__Required__: Yes<br/>
<br/>
The content of the compose file that describes application to create.

____
### RepositoryCredential
__Type__: [RepositoryCredential](sfclient-v56-model-repositorycredential.md) <br/>
__Required__: No<br/>
<br/>
Credential information to connect to container repository.
