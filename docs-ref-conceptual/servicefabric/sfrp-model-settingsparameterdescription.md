# SettingsParameterDescription

Describes a parameter in fabric settings of the cluster.

## Properties
| Name | Type | Required |
| --- | --- | --- |
| [name](#name) | string | Yes |
| [value](#value) | string | Yes |

____
### name
__Type__: string <br/>
__Required__: Yes<br/>
<br/>
The parameter name of fabric setting.

____
### value
__Type__: string <br/>
__Required__: Yes<br/>
<br/>
The parameter value of fabric setting.
