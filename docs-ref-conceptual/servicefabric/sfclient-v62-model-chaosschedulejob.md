# ChaosScheduleJob

Defines a repetition rule and parameters of Chaos to be used with the Chaos Schedule.

## Properties
| Name | Type | Required |
| --- | --- | --- |
| [`ChaosParameters`](#chaosparameters) | string | No |
| [`Days`](#days) | [ChaosScheduleJobActiveDaysOfWeek](sfclient-v62-model-chaosschedulejobactivedaysofweek.md) | No |
| [`Times`](#times) | array of [TimeRange](sfclient-v62-model-timerange.md) | No |

____
### `ChaosParameters`
__Type__: string <br/>
__Required__: No<br/>
<br/>
A reference to which Chaos Parameters of the Chaos Schedule to use.


____
### `Days`
__Type__: [ChaosScheduleJobActiveDaysOfWeek](sfclient-v62-model-chaosschedulejobactivedaysofweek.md) <br/>
__Required__: No<br/>
<br/>
Defines the days of the week that a Chaos Schedule Job will run for.

____
### `Times`
__Type__: array of [TimeRange](sfclient-v62-model-timerange.md) <br/>
__Required__: No<br/>
<br/>
A list of Time Ranges that specify when during active days that this job will run. The times are interpreted as UTC.
