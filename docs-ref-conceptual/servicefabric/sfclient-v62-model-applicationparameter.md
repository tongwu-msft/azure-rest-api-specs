# ApplicationParameter

Describes an application parameter override to be applied when creating or upgrading an application.

## Properties
| Name | Type | Required |
| --- | --- | --- |
| [`Key`](#key) | string | Yes |
| [`Value`](#value) | string | Yes |

____
### `Key`
__Type__: string <br/>
__Required__: Yes<br/>
<br/>
The name of the parameter.

____
### `Value`
__Type__: string <br/>
__Required__: Yes<br/>
<br/>
The value of the parameter.
