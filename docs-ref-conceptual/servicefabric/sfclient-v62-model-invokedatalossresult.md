# InvokeDataLossResult

Represents information about an operation in a terminal state (Completed or Faulted).

## Properties
| Name | Type | Required |
| --- | --- | --- |
| [`ErrorCode`](#errorcode) | integer (int32) | No |
| [`SelectedPartition`](#selectedpartition) | [SelectedPartition](sfclient-v62-model-selectedpartition.md) | No |

____
### `ErrorCode`
__Type__: integer (int32) <br/>
__Required__: No<br/>
<br/>
If OperationState is Completed, this is 0.  If OperationState is Faulted, this is an error code indicating the reason.

____
### `SelectedPartition`
__Type__: [SelectedPartition](sfclient-v62-model-selectedpartition.md) <br/>
__Required__: No<br/>
<br/>
This class returns information about the partition that the user-induced operation acted upon.
