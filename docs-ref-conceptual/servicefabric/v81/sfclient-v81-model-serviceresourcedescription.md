| [`properties`](#properties) | [ServiceResourceProperties](sfclient-v81-model-serviceresourceproperties.md) | Yes |

__Required__: Yes<br/>
<br/>

____
### `properties`
__Type__: [ServiceResourceProperties](sfclient-v81-model-serviceresourceproperties.md) <br/>
