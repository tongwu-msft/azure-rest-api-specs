# RemoteReplicatorAcknowledgementStatus v81

Provides details about the remote replicators from the primary replicator's point of view.

## Properties
| Name | Type | Required |
| --- | --- | --- |
| [`ReplicationStreamAcknowledgementDetail`](#replicationstreamacknowledgementdetail) | [RemoteReplicatorAcknowledgementDetail](sfclient-v81-model-remotereplicatoracknowledgementdetail.md) | No |
| [`CopyStreamAcknowledgementDetail`](#copystreamacknowledgementdetail) | [RemoteReplicatorAcknowledgementDetail](sfclient-v81-model-remotereplicatoracknowledgementdetail.md) | No |

____
### `ReplicationStreamAcknowledgementDetail`
__Type__: [RemoteReplicatorAcknowledgementDetail](sfclient-v81-model-remotereplicatoracknowledgementdetail.md) <br/>
__Required__: No<br/>
<br/>
Details about the acknowledgements for operations that are part of the replication stream data.

____
### `CopyStreamAcknowledgementDetail`
__Type__: [RemoteReplicatorAcknowledgementDetail](sfclient-v81-model-remotereplicatoracknowledgementdetail.md) <br/>
__Required__: No<br/>
<br/>
