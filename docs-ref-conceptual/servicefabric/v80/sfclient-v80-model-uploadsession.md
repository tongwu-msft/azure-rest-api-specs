# UploadSession

Information about a image store upload session

## Properties
| Name | Type | Required |
| --- | --- | --- |
| [`UploadSessions`](#uploadsessions) | array of [UploadSessionInfo](sfclient-v80-model-uploadsessioninfo.md) | No |

____
### `UploadSessions`
__Type__: array of [UploadSessionInfo](sfclient-v80-model-uploadsessioninfo.md) <br/>
__Required__: No<br/>
<br/>
When querying upload session by upload session ID, the result contains only one upload session. When querying upload session by image store relative path, the result might contain multiple upload sessions.
