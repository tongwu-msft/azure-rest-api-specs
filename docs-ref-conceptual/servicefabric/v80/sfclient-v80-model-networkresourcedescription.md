| [`properties`](#properties) | [NetworkResourceProperties](sfclient-v80-model-networkresourceproperties.md) | Yes |

____

__Type__: string <br/>
__Required__: Yes<br/>
<br/>

____
### `properties`
__Type__: [NetworkResourceProperties](sfclient-v80-model-networkresourceproperties.md) <br/>
