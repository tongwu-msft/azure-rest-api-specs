Describes extension of a service type defined in the service manifest.

| --- | --- | --- |
| [Key](#key) | string | No |
| [Value](#value) | string | No |

____
### Key
__Type__: string <br/>
__Required__: No<br/>
<br/>
The name of the extension.

____
### Value
__Type__: string <br/>
__Required__: No<br/>
<br/>
The extension value.
