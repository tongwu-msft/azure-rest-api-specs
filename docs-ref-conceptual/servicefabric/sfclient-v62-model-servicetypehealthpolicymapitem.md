# ServiceTypeHealthPolicyMapItem

Defines an item in ServiceTypeHealthPolicyMap.


## Properties
| Name | Type | Required |
| --- | --- | --- |
| [`Key`](#key) | string | Yes |
| [`Value`](#value) | [ServiceTypeHealthPolicy](sfclient-v62-model-servicetypehealthpolicy.md) | Yes |

____
### `Key`
__Type__: string <br/>
__Required__: Yes<br/>
<br/>
The key of the service type health policy map item. This is the name of the service type.

____
### `Value`
__Type__: [ServiceTypeHealthPolicy](sfclient-v62-model-servicetypehealthpolicy.md) <br/>
__Required__: Yes<br/>
<br/>
The value of the service type health policy map item. This is the ServiceTypeHealthPolicy for this service type.
