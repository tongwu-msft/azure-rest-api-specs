# AutoScalingResourceMetricName enum

type: string

Enumerates the resources that are used for triggering auto scaling.

Possible values are: 

  - `cpu` - Indicates that the resource is CPU cores.
  - `memoryInGB` - Indicates that the resource is memory in GB.
