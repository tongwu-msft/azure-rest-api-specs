# PropertyBatchInfo

Information about the results of a property batch.
# Inheritance

The type 'PropertyBatchInfo' is a base type of the polymorphic type model with property 'Kind' as the discriminator.
Depending upon the value of the property the serialized contents on the wire will be one of the derived types listed below.
## Derived Types

| Kind | Derived Type |
| --- | --- | 
| Successful | [SuccessfulPropertyBatchInfo](sfclient-model-successfulpropertybatchinfo.md) |
| Failed | [FailedPropertyBatchInfo](sfclient-model-failedpropertybatchinfo.md) |
