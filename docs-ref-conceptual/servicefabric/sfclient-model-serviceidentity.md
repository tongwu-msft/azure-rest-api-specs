| Name | Type | Required |
| --- | --- | --- |
| [`name`](#name) | string | No |
| [`identityRef`](#identityref) | string | No |

____
### `name`
__Type__: string <br/>
__Required__: No<br/>
<br/>
The identity friendly name.

____
### `identityRef`
__Type__: string <br/>
__Required__: No<br/>
<br/>
