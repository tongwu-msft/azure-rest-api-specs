# SecretValueResourceDescription

This type describes a value of a secret resource. The name of this resource is the version identifier corresponding to this secret value.

## Properties
| Name | Type | Required |
| --- | --- | --- |
| [`name`](#name) | string | Yes |
| [`properties`](#properties) | [SecretValueResourceProperties](sfclient-v72-model-secretvalueresourceproperties.md) | Yes |

____
### `name`
__Type__: string <br/>
__Required__: Yes<br/>
<br/>
Version identifier of the secret value.

____
### `properties`
__Type__: [SecretValueResourceProperties](sfclient-v72-model-secretvalueresourceproperties.md) <br/>
__Required__: Yes<br/>
<br/>
This type describes properties of a secret value resource.
