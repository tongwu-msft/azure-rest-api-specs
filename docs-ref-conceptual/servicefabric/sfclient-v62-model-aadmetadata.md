# AadMetadata

Azure Active Directory metadata used for secured connection to cluster.

## Properties
| Name | Type | Required |
| --- | --- | --- |
| [`authority`](#authority) | string | No |
| [`client`](#client) | string | No |
| [`cluster`](#cluster) | string | No |
| [`login`](#login) | string | No |
| [`redirect`](#redirect) | string | No |
| [`tenant`](#tenant) | string | No |

____
### `authority`
__Type__: string <br/>
__Required__: No<br/>
<br/>
The AAD authority url.

____
### `client`
__Type__: string <br/>
__Required__: No<br/>
<br/>
The AAD client application Id.

____
### `cluster`
__Type__: string <br/>
__Required__: No<br/>
<br/>
The AAD cluster application Id.

____
### `login`
__Type__: string <br/>
__Required__: No<br/>
<br/>
The AAD login url.

____
### `redirect`
__Type__: string <br/>
__Required__: No<br/>
<br/>
The client application redirect address.

____
### `tenant`
__Type__: string <br/>
__Required__: No<br/>
<br/>
The AAD tenant Id.
