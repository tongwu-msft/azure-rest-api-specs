# ContainerLabel

Describes a container label.

## Properties
| Name | Type | Required |
| --- | --- | --- |
| [`name`](#name) | string | Yes |
| [`value`](#value) | string | Yes |

____
### `name`
__Type__: string <br/>
__Required__: Yes<br/>
<br/>

____
### `value`
__Type__: string <br/>
__Required__: Yes<br/>
<br/>
